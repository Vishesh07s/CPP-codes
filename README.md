# 🚀 CPP-codes (LeetCode Solutions)

Welcome to my repository of C++ solutions! This repository serves as a personal collection of the most interesting, challenging, and insightful algorithmic problems I solve—mostly curated from **LeetCode**.

The codes here are written in **LeetCode-ready format** (Class & Member Functions), optimized for time and space complexity.

## 📚 Repository Structure

The repository contains curated problems across various core concepts:

* **Arrays & Two Pointers:** Problems tracking sliding windows, prefix sums, and multi-pointer approaches.
* **Binary Search:** Advanced searching techniques on sorted arrays and search spaces.
* **Data Structures:** Solutions involving Stacks, Queues, Monotonic structures, and Linked Lists.
* **Advanced Algorithms:** Recursion, Backtracking, and Dynamic Programming.

## 🛠️ How to Use These Solutions

Since these codes are formatted specifically for competitive programming platforms like LeetCode:

1. **Browse** through the repository to find the problem you are looking for.
2. **Open** the `.cpp` file to view the optimized logic.
3. **Copy the `Solution` class** and paste it directly into the LeetCode editor to run or submit.

For every problem added here, the focus is on:
1. **Optimal Approach:** Moving from brute-force to the most optimal Time & Space Complexity.
2. **STL Utilization:** Efficient use of C++ Standard Template Library (`std::vector`, `std::unordered_map`, `std::stack`, etc.).
3. **Clean Code:** Keeping the implementation readable and concise.
